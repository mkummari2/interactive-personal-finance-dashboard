# Interactive Personal Finance Dashboard

## Project Overview
This project is an interactive Excel-based personal finance dashboard designed to help users track income, expenses, savings, and financial performance in a simple and visual way.

## Features
- Income and expense tracking
- Monthly financial summary
- Category-wise expense analysis
- Savings monitoring
- Interactive dashboard visuals
- Easy-to-use Excel interface

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Conditional Formatting
- Dashboard Design

## File Included
- `Interactive_Personal_Finance_Dashboard.xlsx`

## How to Use
1. Download the Excel file.
2. Open it in Microsoft Excel.
3. Update income and expense data.
4. Review dashboard charts and summaries.

## Purpose
The goal of this project is to provide a clear and interactive personal finance tracking solution for budgeting and financial decision-making.
